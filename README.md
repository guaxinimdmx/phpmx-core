# PhpMx Core

Framework minimalista para back-ends modernos em PHP, sem dependências externas, com foco em APIs robustas, performance e controle total do ambiente.

---

## 🚀 Destaques

- 🔁 Sistema de rotas sobrescrevível por projeto
- 🎯 Controllers diretos, sem abstrações desnecessárias
- 📦 Servidor de arquivos com suporte a download e headers adequados
- 🔐 JWT integrado com validação simples e segura
- 🔎 Input validation robusto com tipos e mensagens customizadas
- 🧠 DataLayer completo com migrations, drivers e abstrações de banco
- 🖼️ Manipulação de imagens via GD (DImage)
- 🔒 Cifração e codificação com certificado local (CIF)
- 🧩 Modular: organize tudo por Controller, Route, Driver e mais

---

## 📦 Instalação

    composer require phpmx/core
    php vendor/phpmx/core/install

Para ver o servidor rodando

    php mx server
